A
"adduser <user_name>" (синоним команда  "useradd <user_name>" ) - создаёт новую учетную запись пользователя (добавляет в файл  /etc/passwd  учетную запись пользователя). В Debian и Ubuntu команда сразу запрашивает дополнительную информацию о пользователе и создает (запрашивает) пароль. В других дистрибутивах может потербоваться ввести команду  "passwd <user_name>"  чтобы создать пароль для созданной учетной записи.  

E
export ENV=var - это встроенная функция оболочки Bash в Linux, которая позволяет вам устанавливать переменные среды, к которым могут быть доступны дочерние процессы (программы или задачи, запускаемые из текущей оболочки). Например, export EDITOR=nano. 

S
"su" - позволяет активировать суперпользователя root. После ввода команда запросит пароль пользователя root. 

"sudo bash" - предоставляет максимальные права оболочке bash. Активирует права пользователя root в сеансе работы "sudo bash". Для завершения сеанса с максимальными правами достаточно ввести "exit".

"sudo passwd root" - позволяет задать пароль пользователя root в Ubuntu Linux.

T
"tail /var/log/auth.log | grep sudo" - позволяет контролировать получение прав командой sudo. ВНИМАНИЕ!!! В некоторых дистрибутивах журнал аутентификации называется "secure", а не "auth.log" (cм. "Хакинг на Linux". Д. Колесниченко.2022 г. стр. 42.)

U
"useradd <user_name>" - (cиноним команды  "adduser <user_name>" см. описание данной команды).

V
"sudo visudo" - команда позволяет отредактиовать файл /etc/sudoers, в нём хранятся все нужные параметры для настройки команды sudo. (см. "Хакинг на Linux". Д. Колесниченко.2022 г. стр. 41.)
